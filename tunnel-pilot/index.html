Manage SSH tunnels without touching the terminal

🖥️

System Tray Native

Lives in your menu bar (macOS) or system tray (Windows/Linux). No window clutter — just a clean tray icon with quick access to all your tunnels.

⚡

One-Click Toggle

Toggle SSH tunnels on and off directly from the tray menu. Color-coded status indicators show you the state of each tunnel at a glance.

🔐

Secure Authentication

Supports both password and SSH identity file authentication. Passwords are never included in backup exports for enhanced security.

💾

Backup & Restore

Export your tunnel configurations as JSON and import them on another machine. Identity file paths are preserved, passwords require re-entry.

🔔

Desktop Notifications

Get notified when a tunnel connects, disconnects, or encounters an error. Stay informed without constantly checking the tray.

🚀

Launch at Login

Configure Tunnel Pilot to start automatically when you log in. Your tunnels are always ready when you need them.

Installation

1 Download the latest release

Go to the GitHub Releases page and download the build for your platform.

2 Install

# macOS — drag to Applications
# Windows — run the installer or extract the zip
# Linux — extract and run
chmod +x tunnel_pilot
./tunnel_pilot

3 Configure your tunnels

Click the tray icon, select "Settings...", and add your SSH tunnel configurations. Toggle them on/off directly from the tray menu.

1 Prerequisites

# Install Flutter SDK (3.11.1+)
# https://flutter.dev/docs/get-started/install

# Linux only — install system dependencies
sudo apt-get install libayatana-appindicator3-dev libnotify-dev

git clone https://github.com/kalfian/tunnel-pilot.git
cd tunnel-pilot
flutter pub get

# Run in development
flutter run -d macos

# Build for release
flutter build macos

3 Run tests

flutter test
